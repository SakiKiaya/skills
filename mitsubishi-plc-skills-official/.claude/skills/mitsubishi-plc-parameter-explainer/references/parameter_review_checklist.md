# Parameter Review Checklist

## CPU Parameters

- Watchdog timer
- Scan time related settings
- Operation mode after error
- Latch/retentive range
- Clock/time settings
- I/O refresh behavior

## Module Parameters

- Slot number
- Module model
- Start XY address
- I/O points
- Analog range and scaling
- Error detection behavior

## Network Parameters

- IP address
- Subnet mask
- Gateway
- Station number
- Refresh device assignment
- Timeout and retry
- Master/slave role

## Documentation warnings

Use `需查手冊` when exact model-specific behavior is unknown.
Use `推測` when explanation is inferred from parameter name, comments, or surrounding context.
