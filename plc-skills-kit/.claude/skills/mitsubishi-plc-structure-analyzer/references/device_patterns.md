# Device Pattern Notes

Common device prefixes to scan in source text:

- X: input
- Y: output
- M: internal relay
- L: latch relay
- B/W: link relay/register
- D: data register
- R/ZR: file register
- T: timer
- C: counter
- S: step relay
- SM/SD: special relay/register

These prefixes alone are not enough to decide behavior. Use comments, labels, and cross reference for meaning.
