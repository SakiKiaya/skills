# dotnet-enterprise-doc-generator


## v0.8-problem1 Change

`04_event_flow.md` now uses simplified fixed-participant Mermaid sequence diagrams:

- User
- UI
- Handler
- Logic
- Device
- Storage

This avoids horizontal expansion when an event handler calls many methods.


## v0.8-problem2 Change

`05_method_flow.md` now uses `method_purposes.json` generated by `dotnet-method-purpose-analyzer`.

Each method section must include:

- 用途
- 推測依據
- 觸發來源
- 主要責任
- 副作用
- 維護注意事項
- 分析來源
